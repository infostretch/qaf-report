# qaf-dashboard-ui

*Author: Chirag Jayswal, QAF team*


Static assets for the QAF execution dashboard (`index.html`, `js/`, `css/`, `assets/`). Works from any static host or `file://` (with folder picker). When the same origin serves `qaf-report-core` APIs (`/api/health`, `/api/upload`), upload and server-side index features activate.

The **`qaf-serve`** CLI (`qaf-report-core`) defaults to **http://localhost:2612** (override with a port argument, **`QAF_PORT`**, or **`PORT`**).
